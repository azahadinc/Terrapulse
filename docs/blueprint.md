# **App Name**: TerraPulse

## Core Features:

- Interactive 3D Globe: Visualize event hotspots dynamically on a zoomable, pannable 3D earth model.
- Event Filtering by Category: Filter real-time and historical events by predefined categories such as news, weather, or social trends.
- Detailed Event Information Display: Click on any event hotspot on the globe to reveal comprehensive details.
- AI Event Summarizer Tool: Utilize AI to provide concise summaries of detailed event information, making complex data digestible at a glance.
- Historical Timeline Slider: Navigate through past events using an intuitive timeline slider to view trends and incidents over time.

## Style Guidelines:

- Primary color (Dark Blue): '#3679DC' (HSL: 215, 75%, 45%). Chosen for a sophisticated, data-centric, and expansive feel, reflecting the depth of global insights.
- Background color (Near Black/Dark Grayish-Blue): '#131416' (HSL: 215, 10%, 8%). A deep, desaturated background provides optimal contrast for glowing event hotspots and a focused user experience on the globe.
- Accent color (Cyan/Blue-Green): '#47E1E5' (HSL: 185, 70%, 60%). This vibrant, yet cool accent is analogous to the primary blue and is used for highlighting interactive elements, data visualizations, and 'hotspot' indicators.
- Neutral accents: White (#FFFFFF) for primary text and light gray (#A0A0A0) for secondary information, ensuring readability against the dark scheme.
- Headline and body font: 'Inter' (sans-serif) for its modern, highly readable, and neutral appearance, suitable for displaying complex global data clearly.
- Employ minimalist, geometric line-art icons that align with the tech-forward aesthetic and are easily recognizable for event categories and UI controls.
- Prioritize a full-screen interactive 3D globe as the central element, with peripheral, non-intrusive UI panels for filtering, controls, and detailed event information.
- Incorporate smooth transitions and subtle motion for globe rotations, event highlight animations, and filtering actions to enhance user interaction and visual appeal.
